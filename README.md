# 🚀 Benjamin Vollrath — Portfolio Personal

Sitio web de portfolio personal desarrollado con **React + Vite**, desplegado en **GitHub Pages**.

## 🌐 Demo

👉 [benjivoll.github.io/port-web](https://benjivoll.github.io/port-web)

## ✨ Características

- Diseño moderno en **dark mode** con tipografía Space Grotesk & Inter
- **Navbar** fija con scroll suave y links de redes sociales
- **Hero** con efecto typewriter para roles dinámicos
- **Sobre mí** con estadísticas y badges de skills con íconos SVG
- **Proyectos** destacados con tarjetas interactivas y tags de tecnologías
- **Contacto** con links directos a email, LinkedIn y GitHub
- Totalmente **responsivo**

## 🛠️ Stack

| Tecnología | Uso |
|---|---|
| React 19 | UI / Componentes |
| Vite | Bundler / Dev server |
| CSS Vanilla | Estilos y animaciones |
| gh-pages | Despliegue en GitHub Pages |

## 📦 Instalación y uso local

```bash
# Clonar el repositorio
git clone https://github.com/BenjiVoll/port-web.git
cd port-web

# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev
```

## 🚀 Deploy

```bash
npm run deploy
```

Genera el build y lo publica automáticamente en la rama `gh-pages`.
